## 💻 ModernAuthUI – Login & Signup Form

A clean, modern, and responsive login/signup interface featuring dark mode, gradient buttons, and social media integrations — built with HTML, CSS, and JavaScript. Perfect for demonstrating front-end design skills, responsive layouts, and modern UI/UX principles...

## ✨ Features

⚡ Fully responsive design for desktop, tablet, and mobile

🌙 Dark mode toggle for better user experience

📝 Smooth transitions between login and signup forms

🌐 Integrated social media icons (Facebook, Google, GitHub, Apple)

🎨 Modern UI design with gradient buttons, shadows, and rounded elements

🔒 Clean and simple authentication form layout



---

## 🛠️ Technologies Used

HTML5

CSS3 (Flexbox, Grid, Animations, Transitions, Gradients)

JavaScript (for form toggle & dark mode)

Google Fonts (Poppins)

Font Awesome for social icons

## 🚀 How to Run Locally

1. Clone the repository:



git clone https://github.com/roghayem/ModernAuthUI.git

2. Open index.html in any modern browser to view the login/signup form locally.


3. Click the "Sign up" / "Sign in" buttons to toggle between forms.


4. Use the dark mode button to switch themes.
